Github Many-faced Cards

Build entirely customisable github cards to embed on your webpages.

Quick start

The three ways to get started (refer to the guide here).
IFrame only
<iframe src="themesUrl/themes/<themeName>.gmc.min.html" width="100%" frameborder="0"></iframe>
IFrame + js
<div data-gmc-repo="<full_repo_name>"  data-gmc-theme="<theme_name>" frameborder="0"></div>
<script src="http://cdn.rawgit.com/user/repo/tag/gmc.min.js"></script>

<script>
// Call GMC.loadAllCards() when the document has loaded 
window.onload = GMC.loadAllCards
</script>
Include the markup (more efficient)
<head>
    ...
    <link rel="stylesheet" type="text/css" href="<your_theme_style.css>">
    ...
</head>
<body>
    ...
    <iframe data-gmc-repo="<username/repoName>" data-gmc-theme="<themeName>">
      <!-- Markup of the theme -->
    </iframe>
    <script type="text/javascript">/* Possible js code needed by the theme */</script>
    ...
</body>
