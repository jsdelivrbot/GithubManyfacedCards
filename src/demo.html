Github Many-faced Cards

Build entirely customisable github cards to embed on your webpages.

Quick start

The three ways to get started (refer to the guide here).
IFrame only
<iframe src="https://cdn.rawgit.com/tsucres/GithubManyfacedCards/tree/master/dist/themes/<themeName>.gmc.min.html" width="100%" frameborder="0"></iframe>
Loaded with javascript
// in the head or in the beginning of the body
<script src="https://cdn.rawgit.com/tsucres/GithubManyfacedCards/master/dist/gmc.min.js">

// ...

// where you want the card
<script async src="https://cdn.rawgit.com/tsucres/GithubManyfacedCards/master/dist/gmc-loader.min.js" data-gmc-repo="<full_repo_name>" data-gmc-theme="<theme_name>"></script>
Include the markup (more efficient)
<head>
    ...
    <link rel="stylesheet" type="text/css" href="<your_theme_style.css>">
    ...
</head>
<body>
    ...
    <iframe data-gmc-repo="<username/repoName>" data-gmc-theme="<themeName>">
      <!-- Markup of the theme -->
    </iframe>
    <script type="text/javascript">/* Possible js code needed by the theme */</script>
    ...
</body>
