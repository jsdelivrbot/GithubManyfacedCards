Github Many-faced Cards

Build entirely customisable github cards to embed on your webpages. See how to modify (or build from scratch) a template

IFrame only

<iframe src="themesUrl/themes/<themeName>.gmc.min.html" width="100%" frameborder="0"></iframe>

IFrame + js

<iframe data-gmc-repo="<username/repoName>" frameborder="0" data-gmc-theme="gh_pure"></iframe>
      <script type="text/javascript" src="gmc.min.js"></script>

Include the markup (more efficient)

<head>
    ...
    <link rel="stylesheet" type="text/css" href="<your_themes_style.css>">
    ...
    </head>
    ...
    <div data-gmc-repo="<username/repoName>">
      <!-- Markup of the theme -->
    </div>
    <script type="text/javascript">/* Eventual js code needed by the theme */</script>
    ...
